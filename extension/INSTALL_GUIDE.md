# LogiCheck Extension - Installation Guide

## 📦 Manual Installation

### For Google Chrome

1. **Download** the extension package from the LogiCheck website
2. **Extract** the ZIP file to a folder on your computer
3. Open Chrome and navigate to:
   ```
   chrome://extensions/
   ```
4. Enable **Developer mode** (toggle in the top-right corner)
5. Click **Load unpacked**
6. Select the extracted extension folder
7. The LogiCheck extension is now installed! 🎉

### For Microsoft Edge

1. **Download** the extension package from the LogiCheck website
2. **Extract** the ZIP file to a folder on your computer
3. Open Edge and navigate to:
   ```
   edge://extensions/
   ```
4. Enable **Developer mode** (toggle in the left sidebar)
5. Click **Load unpacked**
6. Select the extracted extension folder
7. The LogiCheck extension is now installed! 🎉

## ⚙️ Configuration

1. Click the LogiCheck icon in your browser toolbar
2. Go to **Settings** (gear icon)
3. Enter your **Gemini API Key**
4. Save your settings

## 🚀 Usage

### Analyze Selected Text
1. Select any text on a webpage
2. Right-click and choose **"Analyze with LogiCheck"**
3. View the analysis in the popup

### Quick Analysis
1. Click the LogiCheck icon in the toolbar
2. Paste or type text in the input field
3. Click **Analyze**

## 🔄 Sync with Website

The extension automatically syncs your API key with the LogiCheck website for a seamless experience across platforms.

## 🆘 Troubleshooting

### Extension doesn't appear after installation
- Make sure you selected the correct folder (the one containing `manifest.json`)
- Try refreshing the extensions page
- Restart your browser

### Analysis not working
- Verify your API key is correctly entered in the settings
- Check your internet connection
- Make sure the API key has sufficient quota

### Sync issues
- Try logging out and back in
- Clear browser cache
- Reinstall the extension
