═══════════════════════════════════════════════════════════════════
                      HOW IT WORKS
═══════════════════════════════════════════════════════════════════

    User Action                System Response
    ──────────────────────────────────────────────────
    
    Click 🌙 icon       →      Toggle theme to dark
                        →      Save to chrome.storage
                        →      Update popup instantly
                        →      Broadcast to all tabs
                        →      Update all sidebars
    
    Click ☀️ icon       →      Toggle theme to light
                        →      Save to chrome.storage
                        →      Update popup instantly
                        →      Broadcast to all tabs
                        →      Update all sidebars
    
    Close popup         →      Theme saved in storage
    
    Reopen popup        →      Load theme from storage
                        →      Apply saved theme
    
    Restart browser     →      Theme still persists!
