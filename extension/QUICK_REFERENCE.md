# LogiCheck Lens - Quick Reference

## 🎯 How to Use (Updated Workflow)

### 1️⃣ First Time Setup

```
Click Extension Icon
    ↓
Click "⚙️ Configure API Key"
    ↓
Enter Gemini API Key
    ↓
Click "Save"
    ↓
Done! ✅
```

**Get API Key:** https://aistudio.google.com/app/apikey

---

### 2️⃣ Analyze Text (Every Use)

**Method 1: Context Menu** (Recommended)
```
1. Select any text on webpage
2. Right-click on selected text
3. Click "Analyze with LogiCheck"
4. Sidebar appears with analysis ✅
```

**Method 2: Keyboard Shortcut** (Fast!)
```
1. Select text
2. Press Ctrl+Shift+L (Windows/Linux)
   OR Cmd+Shift+L (Mac)
3. Sidebar appears with analysis ✅
```

## 🐛 Troubleshooting

**Popup shows old design?**
- Reload extension: chrome://extensions/ → Reload

**Context menu missing?**
- Check extension installed and enabled
- Reload extension

**Sidebar not appearing?**
- Check API key configured
- Look for console errors (F12)
- Verify content script loaded

**Logo not showing in sidebar?**
- Verify web_accessible_resources in manifest
- Reload extension
